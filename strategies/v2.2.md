# Strategy v2.2

Date: 2026-03-11

## Intent

`v2.2` is a research-phase refinement of the ETH/USD paper strategy. It keeps deterministic risk controls outside the analyst and focuses on improving trade quality without "fixing" losses by starving the strategy of trades.

## Core Changes

- Higher-timeframe alignment from 1-minute data using 30m/60m/240m trend proxies and EMA slope bias.
- ATR-based volatility regime gating so the strategy avoids both dead tape and chaotic conditions.
- Explicit multi-signal confirmation:
  momentum, higher-timeframe agreement, trend quality, and breakout plus participation.
- Cost-aware expected edge:
  entries must clear estimated round-trip fees and slippage before becoming executable.
- Standardized execution plan on each entry decision:
  requested notional, order type, TIF, reference price, stop, targets, and expected slippage.
- Stop-distance-aware sizing in the risk layer rather than cash-percentage sizing alone.
- More realistic historical replay:
  adverse slippage, per-side fees, participation-based fill checks, partial profit, lock-in stop after partial, and time stops.

## Success Criteria

- Positive 90-day realized PnL and positive average trade bps.
- Trade count remains meaningful:
  the strategy should not pass by collapsing activity to near zero.
- Candidate score must still exceed the active baseline in walk-forward evaluation.

## Refinement Loop

- Run a 90-day walk-forward backtest.
- Send the result to the offline advisor.
- If performance is negative or activity is too low, version a refined policy family and rerun.
- Once the 90-day run is both positive and active enough, run 180-day validation and send that result to the advisor too.
